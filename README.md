# DCMN - React calculator

This is a DCMN frontend coding challenge boilerplate. Just clone the project, install dependencies and you are ready to go!

### Install dependencies
```
yarn
```

### Run Development Server
```
yarn start
```

### Run Tests
```
yarn test
```

or
```
yarn test-watch
```
